《Better》小记

Jan 21, 2019 | 那些书 | 人拜读过 | 284 字 | 1 分钟

《Better》是被称为医生中的思想家的Atul Gawande的一部早期作品，主要描写的是作为医生，如何在日常工作中不断精进，升级跃迁。这本没有读过，但是听了两次。上一次听这本书是刚考完STEP1，这两天路上（感谢世界首堵，洛杉矶的405）又听了一遍。作者最后给出的5点建议需要不断练习：

  1. 用非医学问题去了解你对面的人
    If you ask a question, the machine begins to feel less like a machine
  2. 别抱怨日常工作，反之，准备其他愉快的话题。从最没难度的天气，到最近读的文章，书，有趣的点子，隔壁邻居家的可爱的狗，街角新开的咖啡店。
  3. 数数。什么都可以，锻炼自己的科研思维。医生不一定成为科学家，但是科研思维却能帮助医生更好的思考，判断，运用基础和临床科研成果。
  4. 写文章。没说的，输出倒逼输入。
  5. 不停改变，成为early adopter，不遗余力寻找改变进步的机会。
